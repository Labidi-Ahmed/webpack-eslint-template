# JavaScript App Starter with Webpack and ESLint
This template provides a minimal setup to get JavaScript working with Webpack and includes essential ESLint rules with the Airbnb style  for maintaining code quality.


### Features

- **Babel Loader**: Transpiles ES6+ JavaScript to ES5.
- **HTMLWebpackPlugin**: Simplifies the creation of HTML files to serve Webpack bundles.
- **WebpackDevServer**: Provides a simple web server with live reloading.
- **Asset Resource Loader**: Manages the loading and processing of asset files.
- **Source Maps**: Assists in debugging by mapping compressed code back to its original source.
- **Reset CSS**: Standardizes browser rendering of HTML elements to provide consistency across different browsers.
- **ESLint with Airbnb Style Guide**: Ensures code quality and consistency with Airbnb's coding standards.

## Usage

Install Dependencies

```
  npm  install
```

Run Dev Server (Port 3000)

```
  npm run dev
```

## Build for production

```
npm run build
```
